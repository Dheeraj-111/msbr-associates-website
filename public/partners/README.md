# Partner Photographs

Place **real** partner photographs in this folder. The site references these files:

- `mayur-sharma.jpg`
- `bharat-rachwani.jpg`
- `pranjal-chipped.jpg`
- `prachi-tale.jpg`
- `jayant-doliya.jpg`

Recommended: square (1:1), at least 400×400px, JPG/WebP, optimized for web.

> Do **not** use AI-generated or stock faces. Until a real photo is added, the UI
> automatically falls back to a gold monogram avatar — no broken images.

To change a filename, update `src/data/partners.ts`.
